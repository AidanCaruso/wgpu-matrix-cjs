Function addScaled

  • addScaled(a: ArrayLike, b: ArrayLike, scale: number, dst?: ArrayLike): vec4.default

  • Adds two vectors, scaling the 2nd; assumes a and b have the same dimension.

    Returns

    A vector that is the sum of a + b * scale.

    Parameters

    • a: ArrayLike

      Operand vector.

    • b: ArrayLike

      Operand vector.

    • scale: number

      Amount to scale b

    • Optional dst: ArrayLike

      vector to hold result. If not new one is created.
